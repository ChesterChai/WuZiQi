### WuZiQi

WuZiQi是一款可以选择棋盘大小，以及AI难度的五子棋游戏，AI引擎使用的是github上的开源项目[EGO AI](https://github.com/tangyan02/ego)，并修改了其中的一小段代码以适应本项目。
